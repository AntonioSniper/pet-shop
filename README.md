## О Laravel

Laravel — это фреймворк для веб-приложений с выразительным, элегантным синтаксисом. Мы считаем, что разработка должна быть приятным и творческим процессом, чтобы приносить истинное удовлетворение. Laravel избавляет от рутины, упрощая выполнение распространённых задач, используемых во многих веб-проектах, таких как:

- [Простой и быстрый механизм маршрутизации](https://laravel.com/docs/routing).
- [Мощный контейнер внедрения зависимостей](https://laravel.com/docs/container).
- Несколько бэкендов для [сессий](https://laravel.com/docs/session) и [кэша](https://laravel.com/docs/cache).
- Выразительный, интуитивный [ORM для баз данных](https://laravel.com/docs/eloquent).
- [Миграции схем БД](https://laravel.com/docs/migrations), независимые от СУБД.
- [Надёжная обработка фоновых задач](https://laravel.com/docs/queues).
- [Трансляция событий в реальном времени](https://laravel.com/docs/broadcasting).

Laravel доступен, мощен и предоставляет все необходимые инструменты для создания больших и надёжных приложений.

## Изучение Laravel

У Laravel самая обширная и подробная [документация](https://laravel.com/docs) и библиотека видеоуроков среди всех современных веб-фреймворков, что позволяет легко начать работу с ним.

Кроме того, [Laracasts](https://laracasts.com) содержит тысячи видеоуроков на самые разные темы, включая Laravel, современный PHP, модульное тестирование и JavaScript. Повысьте свои навыки, изучив нашу обширную видеотеку.

Вы также можете смотреть короткие уроки на реальных проектах на [Laravel Learn](https://laravel.com/learn), где вас проведут через процесс создания приложения на Laravel с нуля, параллельно изучая основы PHP.
